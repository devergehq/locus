# Iterative Depth

2-8 analytical passes through different scientific and professional lenses. Each pass examines the same problem from a fundamentally different perspective, surfacing requirements and risks that single-pass analysis misses.

## Lenses

Each pass uses a different analytical framework:

1. **Systems Thinking** — Feedback loops, emergence, unintended consequences
2. **Information Theory** — Signal vs noise, entropy, compression, data flow
3. **Game Theory** — Incentives, Nash equilibria, strategic behaviour
4. **Evolutionary Biology** — Adaptation, selection pressure, fitness landscapes
5. **Thermodynamics** — Energy flow, entropy, equilibrium, phase transitions
6. **Network Theory** — Nodes, edges, centrality, cascade failures
7. **Cognitive Science** — Mental models, biases, attention, decision-making
8. **Economics** — Scarcity, opportunity cost, market dynamics, externalities

## Process

1. Select 2-4 lenses most relevant to the problem domain
2. Perform a full analytical pass through each lens
3. After each pass, note new insights not visible from previous lenses
4. Synthesise across all passes — where do lenses converge? Where do they contradict?
5. The contradictions are often the most valuable findings
