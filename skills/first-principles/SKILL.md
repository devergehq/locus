# First Principles

Systematic decomposition to foundational truths. Strips away inherited assumptions, conventions, and "best practices" to reveal what's actually required, then reconstructs from verified axioms.

## Process

1. **Identify the claim or approach** — What are we examining?
2. **List all assumptions** — What are we taking for granted?
3. **Challenge each assumption** — Is this actually true? Is it necessary? Where did it come from?
4. **Find the axioms** — What remains when all unverified assumptions are removed?
5. **Reconstruct** — Build back up from only verified truths.

## When to Use

- "Why do we do it this way?" — Challenge inherited architecture
- "Is this actually necessary?" — Question requirements
- "What are the real constraints?" — Separate genuine from assumed constraints
- Root cause analysis — When symptoms keep recurring
