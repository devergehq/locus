# Creative — Templates

Reusable scaffolds for creative work. Each template forces specificity before generation.

## Brief template

```markdown
### Brief

**What is being made:**
<one-sentence description — "a landing page header", "a name for X", "a 90-second script">

**For whom:**
<specific audience — not "users", but "mid-career engineers evaluating infra tools">

**What must it do:**
<the job — "make the reader try the CLI before closing the tab">

**Not what it does:**
<anti-goal — "must not feel like every other AI-infra landing page">

**Constraints:**
- <format limit>
- <exclusion list>
- <reference cue: "in the style of X, but not a copy">

**Stopping condition:**
<how I'll know it's done, not just drafted>
```

## Rejection-first template

Before proposing direction, collect rejections:

```markdown
### Rejections

**I don't want:**
1. <specific generic path to avoid>
2. <specific cliché to avoid>
3. <specific tone to avoid>

**I'm tired of:**
<what feels exhausted in this category>

**It must not:**
<constraints that define failure>
```

Propose direction only after rejections are in place.

## Variant template (for fast iteration)

When generating 3-5 variants on a brief:

```markdown
### Variants

**V1 — Safe / Expected:**
<the default answer>

**V2 — One constraint tightened:**
<apply one additional constraint and see what changes>

**V3 — Inversion:**
<what does the opposite of the safe answer look like>

**V4 — Reference-led:**
<what would this look like if X creative work did it>

**V5 — Emptied:**
<what gets cut if we remove 30% of the content>
```

Pick 1-2 to develop further. Discard the rest without apology.

## Craft / taste split template

```markdown
### Craft vs Taste

**This is a craft problem:**
<if the direction is right but execution fails — sharpen prose, tighten balance, adjust timing>

**This is a taste problem:**
<if the direction itself is wrong — revisit the brief, not the output>
```

When something feels off, name which is broken. Spending craft time on a taste problem makes the output more polished and no better.
