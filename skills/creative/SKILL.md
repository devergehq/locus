# Creative

Divergent ideation framework for generating maximally diverse solutions. Uses structured creative techniques to break out of conventional thinking patterns.

## Techniques

### Verbalized Sampling
Explore the solution space systematically by generating multiple distinct approaches, then evaluating each on different axes (feasibility, novelty, impact, risk).

### Constraint Inversion
List all assumed constraints, then systematically remove or invert each one. "What if we didn't need X?" "What if Y was the opposite?"

### Analogical Transfer
Find structural parallels from unrelated domains. How would this problem be solved in biology? Architecture? Music? Game design?

### Tree of Thoughts
Branch from the initial problem into multiple reasoning paths. Follow each path 2-3 steps. Evaluate which branches yield the most promising territory.

## When to Use

- Stuck on a design problem with no obvious solution
- Need multiple options to evaluate, not just one approach
- Exploring a new problem space where conventions haven't been established
- Looking for non-obvious connections between concepts
