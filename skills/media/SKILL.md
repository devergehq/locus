# Media

Visual content creation framework for technical diagrams, flowcharts, infographics, and illustrations.

## Capabilities

### Mermaid Diagrams
Flowcharts, sequence diagrams, class diagrams, state diagrams, entity-relationship diagrams, Gantt charts, and other structured visualisations using Mermaid syntax.

### Technical Diagrams
Architecture diagrams, system topology, data flow diagrams, and infrastructure visualisations.

### Infographics
Data-driven visual summaries combining statistics, comparisons, timelines, and explanatory graphics.

### Illustrations
Concept illustrations, header images, and visual metaphors for technical content.

## Output Formats

- Mermaid syntax (rendered by compatible viewers)
- SVG (for diagrams and infographics)
- PNG (for raster output)
- Markdown-embedded (inline mermaid blocks)
